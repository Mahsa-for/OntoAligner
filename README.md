[//]: # (## Installation)

[//]: # (OntoAligner is available on PyPI and can be installed with pip:)

[//]: # (```)

[//]: # (pip install ontoaligner)

[//]: # (```)

[//]: # ()
[//]: # (You can also install the latest version from the source:)

[//]: # (```)

[//]: # (git clone git@github.com:sciknoworg/OntoAligner.git)

[//]: # (pip install ./ontoaligner)

[//]: # (```)
